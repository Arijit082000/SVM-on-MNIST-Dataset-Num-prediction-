# 📌 Project Overview

This project demonstrates handwritten digit recognition using the **MNIST** dataset from **OpenML**. An **SVM with an RBF kernel** is trained to classify images into one of the ten digit classes (0–9).

---

# 🎯 Objectives

- Build a handwritten digit classifier.
- Learn image classification with Scikit-learn.
- Train and evaluate an SVM model.
- Measure classification accuracy.

---

# ✨ Features

- Load MNIST dataset from OpenML
- Train/Test split
- SVM (RBF Kernel) implementation
- Model prediction
- Accuracy evaluation
- Classification report generation

---

# 🛠 Technologies Used

- Python
- Scikit-learn
- Pandas
- Matplotlib

---

# 📊 Dataset

- **Dataset:** MNIST
- **Source:** OpenML
- **Classes:** 10 (Digits 0–9)
- **Features:** 784 pixel values per image (28×28)

---

# 🔄 Workflow

1. Import libraries
2. Download MNIST dataset
3. Prepare features and labels
4. Split into training and testing sets
5. Train SVM model
6. Predict test images
7. Evaluate model performance
